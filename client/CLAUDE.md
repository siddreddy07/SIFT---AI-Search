# CLAUDE.md - AI-Search Engine

## Project Overview
AI-Search Engine is a full-stack application with a Next.js client and Node.js server.

## Tech Stack
- Client: Next.js, TypeScript, Tailwind CSS, shadcn/ui
- Server: Node.js, Express, BullMQ, Redis, MongoDB
- AI: LangChain, OpenAI, Pinecone vector store
- Auth: JWT-based authentication

## Key Conventions
- Follow existing code patterns when adding features
- Server uses CommonJS (require/module.exports)
- Client uses ES modules (import/export)
- Always check both client and server when making changes
